# Rime
我的Rime配置

## 使用

1. 下载[Rime](http://rime.im/)并安装

2. 下载[Rime](https://github.com/zYeoman/Rime/archive/master.zip)并解压

3. 打开Rime的“用户文件夹”，把压缩包里的东西扔进去，选择性覆盖，重新部署即可。

4. 开始打字

## 说明

1. 开启symbols.yaml方便输入特殊符号；
2. 提供英语输入
3. 提供颜文字。

### 按键

`\man`：调出表情帮助

`vv`：调出颜文字
